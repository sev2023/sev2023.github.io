Special Characters

Special HTML characters are characters that have special HTML codes associated with them to ensure that they show up correctly in Web browsers.

Example of commonly used special HTML characters are:

  • Copyringt sign © - &copy;
  • Trademark sign ™ - &trade;
  • Pound sterling £ - &pound;
  • Euro sign € - &euro;

Although CSS enables richer text formatting, there are a few HTML5 tags that are good for adding special formatting to text when using specific CSS is not convinient.
Following are some of these tags

  • Superscript text <sup></sup> - Example for superscript text.
  • Subscript text <sub></sub> - Example for subscript text.
  • Strikethrough text <s></s> - Example for strikethrough text.
  • Monospaced text, preserving spaces and line breaks <pre></pre>
